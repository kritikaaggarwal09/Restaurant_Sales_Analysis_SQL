# 🍽️ Restaurant Sales Analysis | SQL Project

## 📌 Project Description
This project is an interactive SQL-based restaurant analysis project designed to explore restaurant operations using structured query language (SQL). The project focuses on analyzing restaurant data to generate meaningful business insights through queries, reports, and data exploration.

The repository contains datasets, and analytical queries .

This project use data to demonstrate how **SQL can be used to analyze data**, answere real-world questions.

# 📊 Key Business Insights 
- The restaurant generated a total revenue of **277,087,363.70**, reflecting strong overall business performance.
- A total of **9,794 menu items** were sold during the analysis period.
- The **Asian category** emerged as the most popular cuisine category among customers.
- The average order value was approximately **2,133**, indicating a healthy customer spending pattern.
- All menu items were sold at least once, meaning there were **no unsold items** in the dataset.
- The restaurant generated an average daily revenue of approximately **129,006**.
- The peak ordering hour was observed at **12:00 PM**, highlighting lunchtime as the busiest period of the day.
- The **three least sold menu items** were: - Chicken Tacos - Cheese Lasagna - Cheese Quesadillas
- The **top three best-selling menu items** were: - Hamburger - Edamame - Korean Beef Bowl

##  🚀 SQL Analysis & Queries

### 1 what is the total revenue generated? 
```sql
select sum(menu_items.price*order_details.order_id) as total_revenue
from menu_items join order_details
on order_details.item_id = menu_items.menu_item_id;
```

### 2 what are the top 5 most sold menu items
```sql
select menu_items.item_name,count(order_details.order_id) as order_sold
from order_details
join menu_items
on order_details.item_id = menu_items.menu_item_id
group by menu_items.item_name
order by order_sold desc limit 5;
```
##  3 what is the average price of an item in each category?
```sql
select category, avg(menu_items.price) as avg_price
from menu_items
group by category
order by avg_price desc;
```

## 4 what is the total revenue generated by each food category
```sql
select menu_items.category, sum(menu_items.price) as total_revenue
from order_details
join menu_items
on order_details.item_id=menu_items.menu_item_id
group by menu_items.category
order by total_revenue desc;
```

## 5 how many orders were placed on a specific month
```sql
select month(order_details.order_date) as months,count( distinct order_details.order_id) 
from order_details
group by months
order by months desc;
```

## 6) what is the most exp item on menu?
```sql
select item_name,price
from menu_items
order by price desc limit 1;
```

## 7) what are the most popular category?
```sql
select menu_items.category, count(order_details.item_id) as total_orders
from order_details
join menu_items
on menu_items.menu_item_id = order_details.item_id
group by menu_items.category
order by total_orders desc limit 1;
```

## 8) how many distinct menu items were sold in total?
```sql
select count(distinct order_details.item_id) as distinct_items_sold
from order_details
join menu_items on menu_items.menu_item_id = order_details.item_id;
```

## 9) what is the avg order value?
```sql
select avg(order_id) AS avg_order_value
from order_details;
```

## 10) what is the items that has not been sold?
```sql
select order_details.item_id,menu_items.item_name
from order_details
left join menu_items
on order_details.item_id = menu_items.menu_item_id
where order_details.item_id <> menu_items.menu_item_id;
```

## 11 what is the daily total revenye (sales pr day)? 
```sql
select day(order_date) as order_day ,sum(menu_items.price) as daily_total_revenue
from menu_items join order_details
on order_details.item_id = menu_items.menu_item_id
group by day(order_date)
order by  order_day asc;
```


## 12)  What is the peak hour for orders?
```sql
select hour(order_details.order_time) as hours, count(order_details.order_id) as orders
from order_details
group by hours
order by orders desc limit 1;
```

## 13) What are the average daily orders?
```sql
select day(order_details.order_date), avg(order_id) as avg_order 
from order_details
group by day(order_date) 
order by day(order_date) asc;
```

## 14) What is the total revenue for the first quarter?
```sql
select quarter(order_date), sum(menu_items.price) as total_revenue
from menu_items
join order_details
on menu_items.menu_item_id = order_details.item_id
group by quarter(order_date)
order by total_revenue;
```

## 15) How many orders have more than one item?
```sql
select count(*) as orders
from (
select order_id from order_details
group by order_id having count(order_id)>1) as all_order;
```     

## 16)  What is the total number of unique customers ?
## -- (assuming each order_id is a unique customer)?
```sql
select count(distinct order_details.order_id) as no_of_customers
from order_details;
```

## 17) What is the average price of an item in the 'American' category?
```sql
select avg(menu_items.price) as  avg_american_item_price
from menu_items
where category = 'American';
```

## 18) Which item has the highest price in the 'Asian' category?
```sql
select item_name,price
from menu_items
where category = 'Asian'
order by  price desc limit 1;
```

## 19) What is the average number of items per order?
```sql
select order_id, count(item_id)as item_per_order
from order_details
 group by order_id;
```

## 20)  How many menu items are there in each category?
```sql
select menu_items.category,count(item_name) as total_items
from menu_items
group by category
order by total_items desc;
```

## 21? What is the total number of menu items sold?
```sql
select count(item_id) as total_menu_items_sold
from order_details;
```

## 22) Which order has the highest total value?
```sql
select order_details.order_id, sum(menu_items.price) as total_order_value
from order_details
join menu_items
on menu_items.menu_item_id = order_details.item_id
group by order_details.order_id
order by total_order_value desc limit 1;
```

## 23) What are the top 3 least sold menu items?
```sql
select menu_items.item_name,
count(order_details.item_id) as times_sold
from menu_items
 join order_details
on menu_items.menu_item_id = order_details.item_id
GROUP BY  menu_items.item_name
ORDER BY times_sold ASC
LIMIT 3;
```
